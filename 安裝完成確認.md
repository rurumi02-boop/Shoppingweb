## 下一步操作

### 1. 驗證所有依賴

在 PowerShell 中執行：

```powershell
# 驗證 Django
python -c "import django; print('✅ Django 版本：', django.get_version())"

# 驗證 Firebase
python -c "import firebase_admin; print('✅ Firebase Admin SDK 已安裝')"

# 查看所有已安裝的套件
pip list | findstr -i "django firebase pymysql"
```

### 2. 如果使用 MySQL，配置 settings.py

如果您的專案需要使用 MySQL（除了 Firebase），在 `CampingData/settings.py` 的最上方添加：

```python
import pymysql
pymysql.install_as_MySQLdb()
```

**注意**：如果主要使用 Firebase，這一步可以跳過。

### 3. 設定 Firebase（重要）

1. **取得 Firebase 憑證**
   - 前往 [Firebase Console](https://console.firebase.google.com/)
   - 創建或選擇專案
   - 前往「專案設定」→「服務帳號」
   - 點擊「產生新的私密金鑰」
   - 下載 JSON 檔案

2. **放置憑證檔案**
   - 將下載的 JSON 檔案重命名為 `firebase-credentials.json`
   - 放在專案根目錄：`C:\CampingData\firebase-credentials.json`

3. **驗證 Firebase 設定**
   ```powershell
   python manage.py check
   ```

### 4. 執行資料庫遷移（如果需要）

```powershell
# 如果使用 Firebase，這步可能不需要
# 但如果 Django 需要初始化，可以執行：
python manage.py migrate
```

### 5. 啟動開發伺服器

```powershell
python manage.py runserver
```

## 常見問題

### Q: 如果 Django 或 firebase-admin 未安裝？
**A:** 
```powershell
pip install -r requirements.txt
```

### Q: Firebase 憑證檔案放在哪裡？
**A:** 
放在專案根目錄：`C:\CampingData\firebase-credentials.json`

### Q: 需要同時使用 MySQL 和 Firebase 嗎？
**A:** 
- 如果主要使用 Firebase，不需要配置 MySQL
- 如果需要 MySQL，在 `settings.py` 中添加 `pymysql.install_as_MySQLdb()`
