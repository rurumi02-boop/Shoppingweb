# 安裝專案依賴 - 解決方案

## 問題說明

錯誤訊息：
```
ERROR: Could not open requirements file: [Errno 2] No such file or directory: 'requirements.txt'
```

**原因**：您當前在 `C:\Users\rurum` 目錄，但 `requirements.txt` 文件在專案目錄 `C:\CampingData` 中。

## 解決方法

### 方法一：切換到專案目錄（推薦）

在 PowerShell 中執行：

```powershell
# 切換到專案目錄
cd C:\CampingData

# 確認文件存在
dir requirements.txt

# 安裝依賴
pip install -r requirements.txt
```

### 方法二：使用完整路徑

在當前目錄直接使用完整路徑：

```powershell
pip install -r C:\CampingData\requirements.txt
```

### 方法三：使用自動化腳本

雙擊執行專案目錄中的 `快速安裝.bat`，腳本會自動處理。

## 完整步驟

1. **切換到專案目錄**
   ```powershell
   cd C:\CampingData
   ```

2. **確認文件存在**
   ```powershell
   dir requirements.txt
   ```
   應該顯示 `requirements.txt` 文件

3. **安裝依賴**
   ```powershell
   pip install -r requirements.txt
   ```

4. **（可選）升級 pip**
   如果看到 pip 有新版本提示，可以執行：
   ```powershell
   python.exe -m pip install --upgrade pip
   ```

## 常見問題

### Q: 還是找不到文件
**A:** 
- 確認您在正確的目錄：執行 `pwd`（PowerShell）或 `cd`（CMD）查看當前目錄
- 確認文件存在：執行 `dir requirements.txt` 或 `ls requirements.txt`

### Q: 安裝時出現權限錯誤
**A:** 
- 使用 `pip install --user -r requirements.txt`（安裝到使用者目錄）
- 或以系統管理員身分執行 PowerShell

### Q: 網路連接問題
**A:** 
- 檢查網路連接
- 嘗試使用國內鏡像源：
  ```powershell
  pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```
