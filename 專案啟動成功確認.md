## 重要發現

### Python 版本問題

系統中有兩個 Python 版本：
- Python 3.14.0（在 PATH 前面）
- Python 3.11.0（已安裝）

**解決方法**：使用 `py -3.11` 來明確指定使用 Python 3.11

## 運行專案的方法

### 方法一：使用 py launcher（推薦）

```powershell
# 啟動開發伺服器
py -3.11 manage.py runserver
```

### 方法二：調整 PATH（永久解決）

1. 找到 Python 3.11 路徑：
   ```powershell
   py -3.11 -c "import sys; print(sys.executable)"
   ```

2. 調整環境變數：
   - 將 Python 3.11 的路徑移到 Python 3.14 之前
   - 或刪除 Python 3.14 的路徑

3. 重新開啟 PowerShell
