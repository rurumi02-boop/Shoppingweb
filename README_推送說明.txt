========================================
  推送 CampingData 到 GitHub
========================================

⚠️ 系統未安裝 Git，請選擇以下方案之一：

【方案一】安裝 Git（推薦）
1. 下載：https://git-scm.com/download/win
2. 安裝時勾選 "Add Git to PATH"
3. 重新啟動命令提示字元
4. 執行：快速推送.bat

【方案二】使用 GitHub Desktop（最簡單）
1. 下載：https://desktop.github.com/
2. 登入 GitHub 帳號
3. File → Add Local Repository
4. 選擇 C:\CampingData
5. Publish repository

【方案三】使用網頁上傳
1. 前往：https://github.com/new
2. 創建倉庫：CampingData
3. 點擊 "uploading an existing file"
4. 拖放文件上傳

詳細說明請查看：推送解決方案.md
