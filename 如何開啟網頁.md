# 🌐 如何開啟網頁 - 快速指南

## 📋 三步驟開啟網頁

### 步驟 1：確認 Firebase 憑證

在啟動前，確認 Firebase 憑證檔案存在：

```powershell
Test-Path C:\CampingData\firebase-credentials.json
```

應該回傳 `True`。如果不存在，請參考 `FIREBASE_MIGRATION.md` 設定。

---

### 步驟 2：啟動開發伺服器

在 **PowerShell** 或 **VS Code 終端機**中執行：

```powershell
# 切換到專案目錄
cd C:\CampingData

# 啟動開發伺服器
py -3.11 manage.py runserver
```

**成功啟動後會看到：**
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

> 💡 **提示**：如果看到遷移警告，這是正常的，可以忽略。

---

### 步驟 3：在瀏覽器中開啟

1. **開啟瀏覽器**（Chrome、Edge、Firefox 等）
2. **在網址列輸入**：`http://127.0.0.1:8000/`
3. **按 Enter**，即可看到網站首頁！

---

## ⚠️ 常見問題

### 問題 1：無法啟動伺服器

**錯誤訊息**：`'py' 不是內部或外部命令`

**解決方法**：
- 確認已安裝 Python 3.11
- 或使用完整路徑：`python manage.py runserver`

---

### 問題 2：找不到 Firebase 憑證

**錯誤訊息**：`⚠️ 警告：未找到 Firebase 憑證檔案`

**解決方法**：
1. 前往 [Firebase Console](https://console.firebase.google.com/)
2. 下載服務帳號憑證 JSON 檔案
3. 重命名為 `firebase-credentials.json`
4. 放在專案根目錄：`C:\CampingData\firebase-credentials.json`

---

### 問題 3：網頁無法開啟

**檢查項目**：
1. 確認伺服器已成功啟動（看到 "Starting development server"）
2. 確認網址正確：`http://127.0.0.1:8000/`（不是 `https`）
3. 確認沒有防火牆阻擋
4. 嘗試重新啟動伺服器

---

## 🛑 停止伺服器

當需要停止開發伺服器時：

1. **在終端機視窗中**按 `Ctrl + C`
2. 或直接關閉終端機視窗

---

## 🔄 重新啟動

下次要開啟網頁時，重複步驟 2 和步驟 3 即可。
